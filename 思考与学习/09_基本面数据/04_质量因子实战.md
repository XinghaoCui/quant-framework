# 质量因子实战 (ROE / 毛利率)

> "质量" = 公司本身赚钱的能力。
> 巴菲特: "用合理的价格买好公司, 比用便宜价格买差公司更重要。"

## 一、质量因子家族

| 因子 | 公式 | 含义 |
|---|---|---|
| ROE | 净利润 / 净资产 | 用净资产赚钱效率 (股东视角) |
| ROA | 净利润 / 总资产 | 用总资产赚钱效率 (含杠杆) |
| 毛利率 | 毛利润 / 营业收入 | 主业"含金量" |
| 净利率 | 净利润 / 营业收入 | 综合赚钱效率 |
| 现金流/利润 | 经营现金流 / 净利润 | 利润是不是真金白银 |
| 资产周转率 | 营业收入 / 总资产 | 资产的"勤奋度" |

直觉: 所有质量指标都是越高越好, 跟"值越大越看多" 约定天然一致 (不用倒数)。

## 二、为什么质量因子能挣钱

经济学解释:
- 持续高 ROE = 强护城河 (Buffett's Moat): 别的公司进不来, 利润长期稳定
- 高毛利率 = 议价能力强: 客户离不开, 不打价格战
- 高现金流转换 = 利润可信: 不是会计美化出来的

学术证据:
- Novy-Marx (2013): "The Other Side of Value" 证明高毛利率因子能产生显著 alpha, 跟价值因子相关性低 (是独立信号)
- Asness, Frazzini, Pedersen (2019): QMJ (Quality Minus Junk) 因子在全球市场都有效

质量因子有个好处: 它不依赖"市场重新估值" (像价值因子那样), 而是依赖"好公司持续生钱", 所以持有时间越长越占便宜。

## 四、ROE 的 DuPont 分解 (高级)

ROE 的经典分解:
$$
\text{ROE} = \frac{\text{净利润}}{\text{净资产}} = \frac{\text{净利润}}{\text{营收}} \times \frac{\text{营收}}{\text{总资产}} \times \frac{\text{总资产}}{\text{净资产}}
$$
$$
\text{ROE} = \text{净利率} \times \text{资产周转率} \times \text{权益乘数}
$$

三个维度的提升路径完全不同:
- 高净利率: 强品牌、强护城河 (茅台、苹果)
- 高资产周转率: 运营效率高 (沃尔玛、低毛利零售)
- 高权益乘数 (= 杠杆): 加杠杆放大 (银行、地产)

警惕: 如果一家公司 ROE 高完全靠加杠杆, 那是危险的。 同样 ROE 20% 的两家公司, 一家靠高毛利, 一家靠 5 倍杠杆, 前者好得多。

实战中可以构造 "纯利润率驱动的 ROE 因子":
```python
adjusted_roe = roe_factor(prices) - low_leverage_factor(prices) * some_weight
# 或更精细的 DuPont 分解
```

## 五、毛利率因子

毛利率是质量的"上游" 指标:
- 高毛利率 = 主营业务赚钱能力强
- 不依赖费用控制 (净利率会受费用扰动)

```python
from src.factors import gross_margin_factor

gm = gross_margin_factor(prices, use_synthetic=True)
```

Novy-Marx 论文里测试的就是"毛利润 / 总资产" (一个略改的版本), 在 1963~2010 年美股上年化 alpha 4% 左右。

```
多空(Qn-Q1) -0.1773  0.0417 -4.2544  0.3906
```

注意看, 模拟数据上 Q1 (高 ROE) 比 Q5 跑得多, 多空价差是负的。

这跟"高 ROE 应该跑赢" 的直觉相反。 怎么回事?

原因是模拟器里"高 ROE" 跟"价格已涨多" 高度耦合, 短期 20 天里"已涨多" 的票要均值回归, 所以高 ROE 反而跑输。

真实数据上 ROE 通常是正向单调 (Q5 > Q1), 这是几十年的实证。
但对教学来说, 这个反例反而很有价值, 它展示了:
1. 因子方向不是"想当然", 必须靠数据
2. 多因子合成 (ic_weighted) 能自动反向利用
3. 模拟数据的统计性质不能直接搬到真实市场

## 七、质量因子常见陷阱

### 陷阱 1: 周期股的"假质量"
钢铁、煤炭在景气年份 ROE 高达 30%+, 萧条年份 ROE 转负。
直接用 ROE 选股 → 永远在景气顶点买进。
解决: 用 5 年平均 ROE 或 ROE 标准差衡量稳定性。

### 陷阱 2: 商誉污染
并购形成大量商誉 → 净资产虚高 → ROE 偏低 (分母大)。
但商誉是"虚的", 调整后真实 ROE 可能更高。
解决: 用 "ROIC" (投入资本回报率) 代替 ROE, 排除商誉影响。

### 陷阱 3: 一次性损益
某季度收到政府补贴 / 卖资产 / 减值, 短期 ROE 跳变。
解决: 用扣非 ROE (扣除非经常性损益)。
本框架的"net_profit" 已是归母净利润, 可考虑改成"扣非归母" 但 akshare 字段需要额外解析。

### 陷阱 4: 银行 / 保险的 ROE 不可比
金融行业天然高杠杆 (杠杆 10 倍+ 是常态), ROE 一般 12~15%, 看上去不输实业 25%。
解决: 行业内中性化, 不跨行业排序 ROE。

## 八、质量 + 价值的经典组合

业界最经典的因子组合就是质量 + 价值 (QV):

```
质量分 = z(ROE) + z(毛利率) + z(净利率)  (取截面 zscore)
价值分 = z(EP) + z(BP)
综合分 = 0.5 × 质量分 + 0.5 × 价值分
```

巴菲特的"用合理价格买好公司" 就是 QV 的语言版本。

进一步加上动量:
```
综合分 = 1/3 × 质量分 + 1/3 × 价值分 + 1/3 × 动量分
```
这叫 QVM, 是 AQR、Two Sigma 等顶级量化机构的核心模型之一。

本框架的多因子合成策略 (`MultiFactorStrategy`) 支持任意因子合成, 用 `combine="equal"` 就能跑 QVM。

## 九、小结

- 质量因子 = "公司本身好不好", 主要看 ROE、毛利率、净利率
- 跟"值越大越看多" 约定天然一致, 不用倒数
- Novy-Marx (2013) 证明高毛利率是独立 alpha 源
- 模拟数据上 ROE 因子是反向的 (教学样本, 真实数据通常正向)
- 陷阱: 周期股假质量、商誉污染、一次性损益、行业不可比
- 实战 QVM (质量 + 价值 + 动量) 是顶级机构标配
