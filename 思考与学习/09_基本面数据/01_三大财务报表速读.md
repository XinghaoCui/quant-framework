# 三大财务报表速读

> 不需要会做账, 但要看得懂表。
> 这一篇用最快的方式让你"知道每张表在说什么, 怎么从里面提取因子"。

## 一、三大表是什么

上市公司一年披露 4 次财报, 每次都包含**三大表**:

```
┌────────────────────────────────────────────────────────────┐
│  利润表 (Income Statement / 损益表)                          │
│  报告: "这一段时间赚了多少钱"                                  │
│  关键字段: 营业总收入 / 净利润 / 毛利率                        │
├────────────────────────────────────────────────────────────┤
│  资产负债表 (Balance Sheet)                                  │
│  报告: "这个时点拥有什么, 欠什么"                              │
│  关键字段: 总资产 / 总负债 / 股东权益 / 资产负债率              │
├────────────────────────────────────────────────────────────┤
│  现金流量表 (Cash Flow Statement)                            │
│  报告: "这一段时间钱怎么流动的"                                │
│  关键字段: 经营/投资/筹资 现金流                               │
└────────────────────────────────────────────────────────────┘
```

记忆口诀:
- 利润表 = 一段时间的"流水账", 看**赚不赚钱**
- 资产负债表 = 一个时点的"家底", 看**有多少钱**
- 现金流量表 = 一段时间的"现金账", 看**钱去哪儿了**

## 二、利润表 (最常用)

简化结构:
```
营业总收入          (顶部)
  - 营业成本
= 毛利润
  - 销售费用
  - 管理费用
  - 研发费用
  - 财务费用
= 营业利润
  + 投资收益
  + 其他
= 利润总额
  - 所得税
= 净利润             (底部)
  - 少数股东损益
= 归母净利润         (最常用的"净利润")
```

提取的常用因子:
| 因子 | 公式 | 含义 |
|---|---|---|
| **毛利率** | 毛利润 / 营业收入 | 主营业务的"含金量", 高 = 议价能力强 |
| **净利率** | 净利润 / 营业收入 | 综合赚钱效率, 扣完所有费用 |
| **营收增速** | 当期营收 / 去年同期营收 - 1 | 主营增长有多快 |
| **净利增速** | 当期净利 / 去年同期净利 - 1 | 利润增长有多快 |

## 三、资产负债表

简化结构:
```
左边 (资产)              =      右边 (来源)
─────────────────              ─────────────────
流动资产                          流动负债
  现金及等价物                     短期借款
  应收账款                         应付账款
  存货                            ...
非流动资产                        非流动负债
  固定资产                          长期借款
  无形资产                          ...
  商誉                            股东权益 (净资产)
  ...                              股本
                                  资本公积
                                  留存收益
─────────────────              ─────────────────
总资产                    =      总负债 + 股东权益
```

提取的常用因子:
| 因子 | 公式 | 含义 |
|---|---|---|
| **资产负债率** | 总负债 / 总资产 | 杠杆水平, 高 = 风险大 |
| **流动比率** | 流动资产 / 流动负债 | 短期偿债能力, < 1 危险 |
| **每股净资产** | 股东权益 / 总股本 | PB 公式的分母 |
| **ROE** | 净利润 / 股东权益 | 用净资产赚钱效率 |
| **ROA** | 净利润 / 总资产 | 用总资产赚钱效率 |

## 四、现金流量表

简化结构:
```
经营活动现金流量        (主营业务产生的现金, 最重要)
  + 投资活动现金流量     (买卖资产、投资别的公司)
  + 筹资活动现金流量     (借钱、发股票、分红)
= 净增加额
```

为什么现金流重要? **利润可以美化, 但现金不会撒谎**:
- 利润表里"应收账款" 算收入, 但客户没付款就不是真金白银
- 健康公司应该: 经营现金流 > 净利润 (回款好)
- 警惕: 利润持续增长但经营现金流为负 → 可能在"虚增收入"

提取的常用因子:
| 因子 | 公式 | 含义 |
|---|---|---|
| **现金流/净利润** | 经营现金流 / 净利润 | 利润含金量, < 1 警惕 |
| **每股现金流** | 经营现金流 / 总股本 | 类似 EPS 但更"真" |

## 五、最常考的 5 个比率

如果你是新手, 至少记住这 5 个:

| 比率 | 公式 | 经验值 |
|---|---|---|
| **ROE** | 净利润 / 净资产 | 持续 > 15% 算优秀公司 |
| **毛利率** | 毛利润 / 营业收入 | 行业差异大, 同行业看相对 |
| **净利率** | 净利润 / 营业收入 | 制造业 5~10%, 互联网 20%+ |
| **资产负债率** | 总负债 / 总资产 | 制造业 < 60% 为佳 |
| **PE** | 股价 / 每股收益 | 大盘股 10~20 区间, < 10 估值低 |

巴菲特名言: "**长期看, 一家公司的回报率不会高于它的 ROE。**" 这 5 个比率是巴菲特看公司的基础。

## 六、akshare 怎么拿这些数据

本框架用 akshare 的 `stock_financial_abstract` 接口, 一次返回 80+ 财务指标。
完整代码见 `src/data/fundamentals.py`, 但用法就一行:

```python
from src.data import get_fundamental_factor

# 拉 ROE 因子, 已对齐到日频且防未来函数
roe = get_fundamental_factor(
    field="roe",
    symbols=["600519", "000333"],
    trading_dates=prices.index,
    use_synthetic=False,  # True = 用模拟数据, 不联网
)
```

支持的字段:
```
价值类:  eps (每股收益), bvps (每股净资产)
盈利类:  net_profit (归母净利润), revenue (营收)
质量类:  roe, roa, gross_margin (毛利率), net_margin (净利率)
成长类:  revenue_yoy (营收同比), profit_yoy (净利同比)
安全类:  debt_ratio (资产负债率)
```

## 七、报表造假与识别 (进阶但重要)

财报不总是真的。 量化研究里有一类"反操纵" 因子, 专门识别造假:

- **应收账款 / 营业收入** 异常高 → 可能虚增销售
- **经营现金流 / 净利润** 持续 < 1 → 利润含金量低
- **存货周转率** 突然下降 → 可能压库存
- **关联交易** 占比高 → 利润转移可能

A 股有不少经典造假案例 (康得新、康美药业、瑞幸咖啡), 量化研究员会专门做"财报质量评分", 排除疑似造假公司。 本框架未实现, 但你要知道有这回事。

## 八、小结

- 三大表: 利润表 (赚不赚)、资产负债表 (有多少)、现金流量表 (钱怎么流)
- 至少记住 5 个比率: ROE、毛利率、净利率、资产负债率、PE
- "利润可以美化, 现金不会说谎" — 永远交叉验证利润和现金流
- 财报数据有造假风险, 量化研究里有专门的"反操纵" 因子
- 本框架接口: `get_fundamental_factor(field, ...)`, 内部已处理披露日
