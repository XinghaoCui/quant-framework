# 为什么要基本面

> 📚 前置: 已经读过 03 因子策略、07 因子研究方法论, 知道因子和 IC 是什么。 完全零基础先看 [00 章](../00_写给零基础的你/00_读我.md)。

前 8 章用的全是**价格数据** — 开高低收成交量。 这一章引入第二条独立信息流: **公司的财务数据**。

## 一、一个生活化类比

你想约会一个人, 决策依据有两种:
- **看 K 线**: 这个人最近的"颜值走势"、"圈内热度"、"被追走势" — 像技术因子
- **看履历**: 学历、工作、家庭、健康、人品 — 像基本面

聪明人**两者都看**:
- 只看 K 线: 容易追高、跟风, 错过被市场低估的"潜力股"
- 只看履历: 容易固执, 不顾市场情绪硬冲
- **两者结合**: 既找出"好公司", 又挑"好时机"

A 股、美股的量化策略也是同样逻辑。

## 二、技术因子 vs 基本面因子

| 维度 | 技术因子 | 基本面因子 |
|---|---|---|
| 数据来源 | 价量 (开高低收成交量) | 财报 (利润表/资产负债表/现金流量表) |
| 频率 | 日频 (高频还有分钟/秒) | 季频 (一年披露 4 次) |
| 时效性 | 实时 | 滞后 1~3 个月 (披露日规则) |
| 例子 | 动量、反转、波动率 | ROE、PE、PB、毛利率、营收增长 |
| 经济含义 | "市场情绪" | "公司质地" |
| 信号衰减 | 快 (几天到几周) | 慢 (几个月到几年) |
| 容量 | 中等 (高频更小) | 大 (低频低换手) |
| 工程难度 | 低 (拿到价格就完事) | 高 (披露日对齐、字段不齐、未来函数) |

**两者覆盖的"市场失效" 不同, 互相不可替代。**

## 三、为什么前 8 章只用技术因子

简单的原因: **数据好搞**。

价格数据:
- yfinance 一行代码拿到美股
- akshare 一行代码拿到 A 股
- 字段标准 (开高低收量), 没有歧义

财报数据:
- 字段名各家接口不一样 ("营业总收入" vs "营业收入" vs "营收")
- 不同会计准则有差异 (旧准则 vs 新准则)
- 披露日 ≠ 报告期 (容易引入未来函数)
- 季频不是日频, 需要做时间对齐
- 上市公司业绩调整、分红送股等导致"历史数据被改"

为了让前 8 章能"小步快跑、立竿见影", 我们先把基本面这一块挂了起来。 **现在 9 章把这块补上, 仓库就完整了。**

## 四、基本面因子的四大类

业界把基本面因子分成四类:

### 1. 价值因子 (Value)
"这只票现在贵不贵?"
- **PE** (市盈率) = 股价 / 每股收益
- **PB** (市净率) = 股价 / 每股净资产
- **PS** (市销率) = 股价 / 每股营收
- **EV/EBITDA**

直觉: PE 低 = 便宜 = 看多。 学界 Fama-French 三因子里的 **HML** 就是 BP (1/PB) 选股。

### 2. 质量因子 (Quality)
"这家公司本身好不好?"
- **ROE** (净资产收益率) — 用净资产赚多少
- **ROA** (总资产收益率) — 用总资产赚多少
- **毛利率** — 单位营收的"含金量"
- **净利率** — 扣完所有费用之后的赚钱效率

直觉: 高 ROE = 好公司 = 长期看多。 Novy-Marx (2013) 的论文证明高毛利率长期跑赢。

### 3. 成长因子 (Growth)
"这家公司是不是在涨?"
- **营收同比增速**
- **净利同比增速**
- **ROE 同比变化**

直觉: 高速增长 = 未来值钱。 但要警惕"高估值陷阱": 增速高的票 PE 也常常很贵。

### 4. 安全因子 (Safety)
"这家公司会不会暴雷?"
- **资产负债率** — 杠杆高不高
- **流动比率** — 短期偿债能力
- **现金流 / 净利润** — 利润是真金白银还是纸面富贵

直觉: 低杠杆 / 高现金流 = 不容易暴雷 = 看多。

## 五、四类因子怎么配合

业界经典做法叫 **QVMS** (Quality + Value + Momentum + Size) 或 **质量-价值-动量**:

```
质量好的票 (高 ROE)
  +
估值便宜 (低 PE)
  +
近期还在涨 (高动量)
  =
"好东西、还在打折、马上要被发现" 的标的
```

**这就是巴菲特 + 西蒙斯的"科学版" 选股逻辑**。
本框架的多因子合成策略 (07 章 / 05 + 本章) 就是在系统化做这件事。
