# 成长与安全因子

> 价值看"便宜不便宜", 质量看"好不好", 成长看"涨不涨", 安全看"会不会爆"。
> 这一篇把后两类讲完, 让你拥有"基本面四大类" 的完整工具箱。

## 一、成长因子 (Growth)

### 核心指标
| 因子 | 公式 |
|---|---|
| **营收同比增速** | (当期营收 - 去年同期营收) / 去年同期营收 |
| **净利同比增速** | (当期净利 - 去年同期净利) / 去年同期净利 |
| **ROE 同比变化** | 当期 ROE - 去年同期 ROE |
| **环比增速 (QoQ)** | 跟上一季度比 |

直觉: 增速越高 = 未来期望越高 = 看多 (在合理范围内)。

### 用本框架算

```python
from src.factors import revenue_growth_factor, profit_growth_factor

rev_g = revenue_growth_factor(prices, use_synthetic=True)
profit_g = profit_growth_factor(prices, use_synthetic=True)
```

### 为什么不用环比 (QoQ) 而用同比 (YoY)?
A 股很多行业有强季节性:
- 白酒 Q1 (春节) 营收最高
- 农业 Q3 (秋收) 业绩集中
- 零售 Q4 (双 11、圣诞) 业绩集中

环比比较会被季节性干扰严重。 **同比 (跟去年同一季度比) 自动消除季节性**, 更稳。

### 成长因子的陷阱

#### 陷阱 1: 高基数效应
去年 Q1 净利 100 万, 今年 Q1 净利 200 万 → 增速 100%, 很美。
明年 Q1 净利 250 万 → 增速 25%, 数字"下降"了, 实际还在涨。
**直觉**: 成长率有"基数效应", 持续高增速越来越难, 不要拿"增速下降" 当卖出信号。

#### 陷阱 2: 一次性事件污染
公司去年卖了一栋楼, 净利 +1 亿; 今年没卖了, 净利 -8000 万。
**解决**: 用"扣非净利润同比" 而不是"净利润同比"。

#### 陷阱 3: 高增长 + 高估值
最赚钱的票常常是"低估值 + 高增长" (PEG < 1, Quadrant I), 单看"高增长" 会选到"高估值高增长" 的茅台、宁德 — 已经被市场充分定价。
**解决**: 配合价值因子 (PEG = PE / 增速)。

## 二、安全因子 (Safety)

### 核心指标
| 因子 | 公式 | 含义 |
|---|---|---|
| **资产负债率** | 总负债 / 总资产 | 杠杆, 高 = 风险大 |
| **流动比率** | 流动资产 / 流动负债 | 短期偿债, < 1 危险 |
| **速动比率** | (流动资产 - 存货) / 流动负债 | 更严的短期偿债 |
| **利息保障倍数** | EBIT / 利息费用 | 还利息能力 |
| **现金流/利润** | 经营现金流 / 净利润 | 利润真实性 |
| **Altman Z-Score** | 多因子加权综合 | 经典破产预警模型 |

直觉: 低杠杆 / 高流动性 / 高现金流转换 = 安全。

### 用本框架算

```python
from src.factors import low_leverage_factor

ll = low_leverage_factor(prices, use_synthetic=True)
# 内部就是 -debt_ratio, 满足"值越大越看多"约定
```

### 为什么安全因子单独看不强但配进组合很重要

安全因子在"日历上" 不太挣钱:
- 长期看, 适度加杠杆的公司 ROE 更高, 跑赢低杠杆的
- 单独按低杠杆排序选股, 通常打不过等权

但安全因子在 **危机时期** 价值巨大:
- 2008 金融危机: 高杠杆公司大量破产, 低杠杆的活下来
- 2020 疫情: 现金流强的公司没被冲垮
- 2022 加息: 浮动利率债务多的公司利息暴增

**作用**: 安全因子是组合的"保险", 在尾部风险下保护本金。 不是为了"挣得多", 而是为了"亏得少"。

## 三、四大类因子整合: QVGS

把价值 / 质量 / 成长 / 安全 都加进组合就是 **QVGS 模型** (Quality + Value + Growth + Safety):

```
质量 = z(ROE) + z(毛利率) + z(净利率)
价值 = z(EP) + z(BP)
成长 = z(营收同比) + z(净利同比)
安全 = z(-资产负债率) + z(经营现金流/净利润)

综合 = 1/4 × 质量 + 1/4 × 价值 + 1/4 × 成长 + 1/4 × 安全
```

每一类是一个"主题", 综合分对应**全面好的公司**:
- 好生意 (高质量)
- 估值合理 (有价值)
- 还在增长 (有成长)
- 不会暴雷 (够安全)

这就是巴菲特 + 芒格选股法的量化版本。 AQR 的"AAA" 因子集合大致也是这个思路。

## 五、各因子的"性格": 该用 IC 还是用单调性

不同因子有不同的检验方式:

| 因子类型 | 偏好检验 | 原因 |
|---|---|---|
| 价值 (EP/BP) | 多空价差 + 单调 | 估值修复需要时间, 看长期 |
| 质量 (ROE) | IC + 单调 | 信号稳定, 适合横截面排序 |
| 成长 (YoY) | IC | 受基数效应影响, 单调不明显 |
| 安全 (杠杆) | 危机时期单独看 | 平时无效, 危机时关键 |

**做研究时不要一刀切**, 不同因子用不同检验方式。

## 七、小结

- 成长因子 = "涨不涨", 用同比避季节性, 警惕基数效应
- 安全因子 = "会不会爆", 日历表现一般, 危机时关键
- 四大类 (价值 + 质量 + 成长 + 安全) = QVGS, 巴菲特选股法量化版
- 本框架 `all_fundamental_factors` 一键拉全部, 满足看多约定
- 不同因子用不同检验方式, 不要一刀切
- 模拟数据上 ic_weighted 完胜等权 (D vs C 演示)
