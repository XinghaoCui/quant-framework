# 价值因子实战 (EP / BP)

> 价值投资 = "用便宜价格买好东西"。
> 价值因子就是把"便宜不便宜" 量化, 用历史数据验证它能不能挣到 alpha。

## 一、价值因子家族

| 因子 | 全名 | 公式 |
|---|---|---|
| PE | Price-to-Earnings | 股价 / 每股收益 |
| PB | Price-to-Book | 股价 / 每股净资产 |
| PS | Price-to-Sales | 股价 / 每股营收 |
| EV/EBITDA | Enterprise Value / EBITDA | 企业价值 / 息税折旧前利润 |

直觉: 比率越低 = 越便宜 = 看多。

## 二、为什么要用"倒数 EP / BP" 而不是 PE / PB

本框架的约定: 所有因子值越大越看多。

PE 是"价格除以收益", 越大代表越贵。 如果直接用 PE 当因子, 它的方向跟约定相反, 容易混淆。

所以我们用倒数:
$$
EP = \frac{1}{PE} = \frac{\text{每股收益}}{\text{股价}}
$$

- EP 大 = PE 小 = 便宜 = 看多
- 跟其他因子 (ROE 大看多、动量大看多) 方向统一

同理 BP = 1 / PB 是"每股净资产 / 股价"。

Fama-French 三因子模型里的 HML (High Minus Low) 就是按 BP 排序构造的, 学界公认最经典的"价值"因子。

## 三、价值因子能挣钱吗

学术与实证证据:
- Fama & French (1992): BP 高的股票长期跑赢 BP 低的
- 巴菲特: 一辈子的核心策略就是"低估值好公司"
- 量化界: 价值因子在多因子模型里几乎必选

但近 10 年价值因子表现不好:
- 美股 2010~2020: "成长股" 完胜"价值股", FAANG 涨上天
- 主要原因: 低利率环境 + 科技股估值扩张
- 2022 加息后价值因子反弹, 但波动很大

结论: 价值因子是长周期有效的, 但短期可能 5~10 年跑输。 这跟它的经济逻辑有关 (需要"市场重新估值" 才能兑现 alpha, 而市场可以长期非理性)。

## 四、用本框架算 EP / BP

```python
from src.data import generate_synthetic_prices
from src.factors import earnings_yield, book_to_price

prices = generate_synthetic_prices(
    [f"STK{i:02d}" for i in range(30)],
    "2020-01-01", "2024-12-31", seed=42,
)

# EP 因子
ep = earnings_yield(prices, use_synthetic=True)
print(ep.tail(3).round(4))

# BP 因子
bp = book_to_price(prices, use_synthetic=True)
print(bp.tail(3).round(4))
```

`earnings_yield` 内部做的事:
1. 用 `get_fundamental_factor("eps", ...)` 拿 EPS (已防未来函数对齐到日频)
2. 用 `EPS / Price` 算 EP
3. 处理无效值 (除零等)

## 五、跑示例 41 看 IC

`examples/41_价值与质量因子IC.py` 把 EP、BP 跟 7 个因子一起跑 IC:

```
因子                       IC均值       IR      t统计     IC>0占比       样本
----------------------------------------------------------------------
EP                    -0.0980   -0.478   -16.46     28.45%     1188
BP                    -0.0728   -0.355   -12.23     33.92%     1188
ROE                   -0.0655   -0.356   -12.28     36.87%     1188
...
```

注意: 模拟数据上 EP / BP 的 IC 都是负的 (≈ -0.07~-0.10)。

为什么?
- 模拟器里"高 ROE 的票" 价格涨多了 → EPS 高 → EP 也高 (高估值)
- 但价格涨多了, 未来收益率反而低 → forward IC 负
- 这是模拟数据的特定行为, 真实数据上 EP 通常是正 IC (估值修复 alpha)

这里有个重要教学点: 因子方向不是靠直觉拍脑袋, 而是靠数据说话。 即使方向跟直觉反, ic_weighted 模式会自动反向利用 (08 章演示过)。

## 六、价值因子的常见陷阱

### 陷阱 1: 单期 EPS 噪声大
某季度 EPS 突然变 0 (一次性减值), PE 就爆掉。
解决: 用 EPS TTM (过去 4 季度滚动) 平滑噪声。
本框架暂用单季度 EPS, 实战推荐 TTM。

### 陷阱 2: 行业差异巨大
银行 PE 5~10, 互联网 PE 30~80。 跨行业按 PE 选股 = 永远选银行。
解决: 行业内中性化, 先做行业去均值再排序。

### 陷阱 3: "便宜的理由" 可能是基本面恶化
低 PE 公司可能是因为业绩在崩 (PE = 价格 / EPS, EPS 跌得比价格快)。
这叫"价值陷阱" (Value Trap)。
解决: 价值因子配合"质量" 或"动量" 因子, 排除业绩崩盘的票。

### 陷阱 4: 周期股的 PE 反着读
周期股 (煤炭、钢铁) 业绩波动剧烈:
- 行业景气时 EPS 高, PE 极低 → 实际是"高位估值"
- 行业萧条时 EPS 低或负, PE 极高或无意义
解决: 周期股看 P/B 比 P/E 更靠谱。

## 七、价值因子的"反操纵" 升级

业界为对抗陷阱, 通常用更复杂的价值因子:

- EP_growth_adjusted = EP × 净利增速 (用 PEG 思路)
- EV/EBITDA: 用企业价值 + 息税折旧前利润, 消除资本结构差异
- Free Cash Flow Yield = 自由现金流 / 市值, 真金白银的"价值"

本框架只实现了基础 EP/BP, 进阶版你可以自己加。

## 八、价值 vs 成长: 二维分类

学界把所有股票按"价值 - 成长" 维度分类:

```
                成长 (Growth, 营收/利润高增长)
                  ↑
                  |
                  |  Quadrant II           Quadrant I
                  |  (高成长 + 高估值)       (高成长 + 低估值)  ← 量化研究员最爱
                  |  Tesla 2021             (Buffett 抄底)
                  |
低估值 ←──────────┼──────────→ 高估值
(Value)           |               (Growth)
                  |
                  |  Quadrant III          Quadrant IV
                  |  (低成长 + 低估值)        (低成长 + 高估值)
                  |  价值陷阱可能区          → 通常没人选
                  ↓
              不成长
```

最赚钱的区域是 Quadrant I (高成长 + 低估值), 但这种票极少, 找到一只就是几年的 alpha。
学量化的目标就是用因子组合 + 行业判断系统化地找出 Quadrant I 的票。

## 九、小结

- 价值因子 = "便宜"。EP = 1/PE, BP = 1/PB, 跟看多约定方向统一
- 学界 Fama-French 三因子里的 HML 就是 BP
- 近 10 年价值因子表现一般, 长周期仍是经典
- 模拟数据上 EP/BP 的 IC 是负的, 不代表真实数据上, 拿数据说话
- 陷阱: 行业差异、价值陷阱、周期股、EPS 噪声
- 本框架接口: `earnings_yield(prices)`, `book_to_price(prices)`
