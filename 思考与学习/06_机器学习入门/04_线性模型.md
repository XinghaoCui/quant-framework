# 线性模型

> 永远从线性模型开始。它简单、快、可解释，而且在金融里常常打败复杂模型。

## 一、普通线性回归

### 模型
$$
\hat{y} = w_1 x_1 + w_2 x_2 + \dots + w_n x_n + b
$$

预测值是特征的加权和。模型要学的就是权重 $w$ 和截距 $b$。

在量化里：
$$
\text{预测收益} = w_1 \cdot \text{动量} + w_2 \cdot \text{波动率} + \dots
$$

**这其实就是"自动学权重的多因子模型"！** 传统多因子是人定 $w$，线性回归让数据定 $w$。

### 怎么学权重

最小化预测误差的平方和（最小二乘）：
$$
\min_w \sum_i (y_i - \hat{y}_i)^2
$$

sklearn 一行搞定，数学细节不用手算。

### 优点
- **可解释**：每个因子的权重一目了然，知道模型靠什么
- **快**：几万样本毫秒级
- **稳健**：不容易过拟合（参数少）

### 缺点
- 只能学**线性关系**，捕捉不了"动量和波动率的交互"这种非线性
- 对**异常值敏感**（平方惩罚放大极端误差）
- 对**特征量纲敏感**（所以必须先标准化）

## 二、为什么要标准化

线性模型里，如果动量在 [-0.5, 0.5]、波动率在 [0, 100]，
权重会被量纲扭曲。标准化把每个特征变成均值 0、方差 1：

$$
x_{std} = \frac{x - \mu}{\sigma}
$$

本框架自动处理（Pipeline 里的 StandardScaler）：
```python
Pipeline([("scaler", StandardScaler()), ("model", Ridge())])
```

**关键**：scaler 只在训练集 fit，避免数据泄露（见 [03_金融数据的坑](./03_金融数据的坑.md)）。

## 三、正则化：Ridge 和 Lasso

普通线性回归在特征多、相关性高时会过拟合。
正则化通过**惩罚大权重**来抑制过拟合。

### Ridge 回归（L2 正则）
$$
\min_w \sum_i (y_i - \hat{y}_i)^2 + \alpha \sum_j w_j^2
$$

后面那项惩罚"权重平方和"。$\alpha$ 越大，权重被压得越小，模型越简单。

效果：把所有权重**平滑地缩小**，但不会变成 0。
适合：特征都有点用、相关性高的场景。**量化里的默认选择。**

### Lasso 回归（L1 正则）
$$
\min_w \sum_i (y_i - \hat{y}_i)^2 + \alpha \sum_j |w_j|
$$

惩罚"权重绝对值和"。

效果：会把不重要特征的权重**直接压成 0** → **自动特征选择**。
适合：你怀疑很多特征是噪声，想让模型自己挑。

### Ridge vs Lasso 对比

| | Ridge (L2) | Lasso (L1) |
|---|---|---|
| 惩罚项 | 权重平方 | 权重绝对值 |
| 效果 | 缩小所有权重 | 部分权重归零 |
| 特征选择 | 不会 | 会 |
| 适用 | 因子都有用 | 想筛因子 |

### α（正则强度）怎么选
- α=0：退化成普通线性回归（不正则）
- α 太小：正则没效果，可能过拟合
- α 太大：模型太简单，欠拟合
- 用交叉验证（`purged_kfold`）选最优 α

```python
make_model("ridge", alpha=1.0)    # alpha 是正则强度
make_model("lasso", alpha=0.001)
```

## 四、逻辑回归（分类）

预测涨/跌方向时用逻辑回归。它把线性输出通过 sigmoid 压到 [0,1]，解释为"上涨概率"：

$$
P(\text{涨}) = \frac{1}{1 + e^{-(w \cdot x + b)}}
$$

```python
make_model("logistic", task="classification")
```

预测时返回上涨概率，按概率排序选股（`ModelWrapper.predict` 自动用 `predict_proba`）。

## 六、为什么线性模型在量化里这么重要

1. **基准**：任何复杂模型都要先打败线性模型才有意义
2. **可解释**：能向风控、向老板、向面试官解释"为什么买"
3. **稳健**：参数少，不容易在小样本上过拟合
4. **快**：适合大规模因子测试和频繁重训
5. **接近真相**：很多金融关系本来就近似线性

业界大量实盘策略就是**正则化线性模型 + 好特征**，朴实但有效。

## 七、小结

- 线性回归 = 自动学权重的多因子模型
- 必须标准化特征（量纲敏感）
- Ridge（L2）缩小权重，Lasso（L1）做特征选择
- α 控制正则强度，用 CV 选
- 分类用逻辑回归（输出概率，按概率排序）
- **永远先用线性模型打基准**
