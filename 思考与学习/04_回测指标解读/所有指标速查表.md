# 所有回测指标速查表

本框架 `compute_metrics()` 自动计算下面所有指标。

## 收益类

| 指标 | 公式 | 经验值 | 说明 |
|---|---|---|---|
| 累计收益 | `(1+r).cumprod()[-1] - 1` | — | 总共赚了百分之几 |
| 年化收益 | `(1+total)^(252/N) - 1` | 5-20% 可接受，>30% 警惕 | 几何平均，等价稳定增长率 |
| 年化波动率 | `r.std() * √252` | 10-25% 常见 | 衡量收益剧烈度 |

## 风险调整收益类

| 指标 | 公式 | 经验值 | 说明 |
|---|---|---|---|
| **Sharpe** | `(年化-rf) / 年化波动` | >1 可投资，>2 优秀 | 通用风险调整收益 |
| **Sortino** | `(年化-rf) / 下行波动` | 通常比 Sharpe 高 30%+ | 只惩罚亏损波动 |
| **Calmar** | `年化收益 / |MDD|` | >1 优秀 | 收益相对最大痛点 |

## 风险类

| 指标 | 公式 | 经验值 | 说明 |
|---|---|---|---|
| **最大回撤** | `min(net/peak - 1)` | -10% 稳健，-30% 一般 | 历史最痛时刻 |
| 下行波动率 | `√(mean(min(r,0)²)) * √252` | — | 用于 Sortino |

## 交易频率类

| 指标 | 公式 | 经验值 | 说明 |
|---|---|---|---|
| 胜率 | `(r > 0).mean()` | 40-60% | 盈利天数占比 |
| 盈亏比 | `mean(r+) / |mean(r-)|` | 趋势策略 >1.5 | 平均盈利/平均亏损 |
| 平均换手率(单边) | `|Δw|.sum() / 2` | 因策略而异 | 调仓频繁度 |
| 总交易成本 | `换手 × (佣金+滑点)` | <年化收益 1/3 | 否则策略不实用 |

## 几个指标的关系

### 胜率 vs 盈亏比

> 胜率 60% + 盈亏比 0.8 vs 胜率 40% + 盈亏比 1.8

后者其实更赚钱（期望 0.4*1.8 - 0.6*1 = 12% vs 0.6*0.8 - 0.4*1 = 8%）。

**单看胜率没用**，要和盈亏比一起看。趋势策略胜率低（30-40%）但盈亏比高，靠少数大赢覆盖多数小亏。

### Sharpe vs Calmar

Sharpe 用整体波动，Calmar 用极端损失。
两者通常正相关，但不完全一致：
- 高 Sharpe + 低 Calmar：日常稳定但偶尔大亏（卖期权类）
- 低 Sharpe + 高 Calmar：波动大但回撤小（高波低偏的策略）

### Sharpe vs Sortino

Sortino 通常 = 1.2~1.5 × Sharpe。
如果 Sortino 远高于 Sharpe，说明上行波动多（好事，左偏小）。
如果 Sortino 接近 Sharpe，说明上下波动对称。

## 报告里要写哪些

最少应该报告：

| 必报 | 选报 |
|---|---|
| 累计收益 | Sortino |
| 年化收益 | Calmar |
| 年化波动 | 胜率 |
| Sharpe | 盈亏比 |
| 最大回撤 | 换手率 |
| 交易天数 | 总交易成本 |
|  | 与基准的对照 |
