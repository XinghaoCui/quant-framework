# 风险平价 (Risk Parity)

> 业界最主流的"不预测收益" 分仓方法之一。
> 桥水"全天候" 基金 (Bridgewater All Weather) 的内核思想。

## 一、一句话直觉

> **每只股票对组合总风险的贡献应该相等。**

类比: 4 个人合伙开公司, 每个人**承担同样多的责任** (即使出资不同) → 这就是"风险平价"。

数学上, "风险贡献" 指每只票对组合方差的边际贡献。
风险平价就是让 **RC₁ = RC₂ = ... = RCₙ = σ_p / N**。

## 二、为什么不是简单的"反波动率"

很多人把"反波动率" 等同于"风险平价", 这是常见误区。

**反波动率 (Inverse Volatility)**:
$$
w_i \propto \frac{1}{\sigma_i}
$$
只用各只票自己的波动, **完全忽略相关性**。

**风险平价 (Risk Parity)**:
要求每只票的**完整风险贡献** (含相关性) 相等:
$$
RC_i = w_i \cdot \frac{(\Sigma \mathbf{w})_i}{\sigma_p} = \frac{\sigma_p}{N}
$$

差异:
- 票 A 和 B 高度相关 → A 提高 1% 权重, 会让 B 的风险贡献也增加 (因为它俩同涨同跌)
- 反波动率看不见这一点, 风险平价能看见

实战上, 反波动率是风险平价的**一阶近似** (假设相关性 = 0)。 在相关性较低的股票池里, 两者很接近。

## 三、风险贡献的数学

定义:
$$
\sigma_p^2 = \mathbf{w}^T \Sigma \mathbf{w}
$$

对 wᵢ 求偏导:
$$
\frac{\partial \sigma_p}{\partial w_i} = \frac{(\Sigma \mathbf{w})_i}{\sigma_p}
$$

这是"票 i 的边际风险" — 它增加 1 单位权重, 组合波动会增加多少。

把边际风险乘上 wᵢ, 得到"票 i 的总风险贡献":
$$
RC_i = w_i \cdot \frac{(\Sigma \mathbf{w})_i}{\sigma_p}
$$

可以证明: **所有 RCᵢ 之和 = σ_p**。 风险平价就是让每只票均分这 σ_p。

## 四、怎么求解

没有解析解, 用数值优化求:

```python
# 目标: 让所有 RCᵢ 之间的差异最小
def objective(w):
    rc = w * (Sigma @ w) / np.sqrt(w @ Sigma @ w)
    return np.sum((rc - rc.mean()) ** 2)

# 用 SLSQP, sum(w)=1, w>=0
```

完整实现见 `src/portfolio/optimizer.py:risk_parity_weights`。

收敛通常很快 (<1 秒 / 几十只票)。 真要做几百只票, 推荐用 `riskparityportfolio` 库 (C++ 后端)。

## 五、用法

```python
from src.portfolio import risk_parity_weights

returns = prices.pct_change().dropna()
res = risk_parity_weights(returns)
print(res.weights.round(4))
```

验证"真的平价":
```python
w = res.weights.values
cov = returns.cov().values
port_vol = np.sqrt(w @ cov @ w)
rc = w * (cov @ w) / port_vol
print(rc)  # 应该所有值近似相等
```

`examples/30_组合优化对比.py` 验证过, 6 只票的 risk_parity 权重:

```
              risk_parity
STK00          0.1874
STK01          0.1635
STK02          0.1406
STK03          0.1869
STK04          0.1761
STK05          0.1455
```

波动越大的票权重越低, 但比反波动率"更精细" (考虑了相关性)。

## 六、风险平价的两大优点

### 优点 1: 不需要预测收益, 比等权稳
跟最小方差一样, 风险平价**不依赖期望收益**, 避开了"μ 估不准" 的灾难。

### 优点 2: 比最小方差更分散
最小方差经常**集中到几只波动低的票**, 风险平价不会。
它要求每只票都"出一份力", 自然分散。

机构投资人偏爱风险平价的原因, 主要是这两点。 **桥水的"全天候"基金**长期年化 12% 左右, 夏普 0.7~1.0, 是风险平价最有名的实战。

## 七、风险平价的扩展: 风险预算

风险平价是"每只票均分风险", 推广一下, 你可以让"每只票按设定比例分风险":

$$
RC_i = b_i \cdot \sigma_p \quad \text{其中} \quad \sum b_i = 1
$$

bᵢ 是你设定的"风险预算":
- b = (1/N, ..., 1/N) → 风险平价
- b = (0.6, 0.4, 0, ..., 0) → 60% 风险给前两只票
- b = (0.5 股票, 0.3 债券, 0.2 商品) → 跨资产配置

这就是**风险预算 (Risk Budgeting)**, 比风险平价更灵活。 本框架暂未实现, 但思路上是 risk_parity 的小幅推广。

## 八、坑

### 坑 1: 不分资产类别用风险平价很有限
经典风险平价用在**跨资产** (股 / 债 / 商品 / 现金) 上效果最好, 因为不同资产类别相关性低。
全是股票时, 相关性都 0.5+, 风险平价的效果跟反波动率差不多。

### 坑 2: 数值优化偶尔不收敛
RC 函数是非凸的, SLSQP 可能陷入局部最优。
本框架默认 maxiter=5000, 通常没问题, 极端情况下可以加 max_iter 或换初值。

### 坑 3: 看似平价, 实则受 Σ 误差影响
风险贡献依赖 Σ 估计。 同样的 Σ 噪声问题适用。
解决方案同上: shrinkage、长 lookback。

## 九、什么时候用风险平价

**适用**:
- 跨资产配置 (股+债+商品)
- 没有可靠收益预测、追求稳定
- 希望"每只资产都参与" (避免最小方差的集中)

**不适用**:
- 你有非常强的选股信号 (浪费)
- 票池规模很大且全是股票 (跟反波动率没区别, 不如直接用反波动率快)

## 十、小结

- 风险平价 = 每只票对组合风险贡献相等
- ≠ 反波动率 (反波动率只看单只波动, 忽略相关性)
- 数值求解, scipy 几秒搞定
- 优点: 不需要 μ + 自然分散
- 桥水"全天候" 基金的内核, 跨资产时威力最大
- 接口: `risk_parity_weights(returns)` 或 `OptimizedStrategy(scores, optimizer="risk_parity")`
