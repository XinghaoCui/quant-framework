# 思考与学习 - 总览

这个文件夹记录了搭建这个量化框架时的**思考过程**，目的是让你（崔兴昊）能：
1. 理解每个模块"为什么这么写"，而不只是"它是什么"
2. 在面试时能讲清楚设计取舍
3. 自己往里加新策略/因子时有方法论

## 推荐阅读顺序

```
01_架构与设计/  ──→  02_核心术语/  ──→  03_因子与策略/
                                              ↓
05_扩展指南/   ←──  04_回测指标解读/  ←─────┘
```

## 学习建议

- 先跑通 `examples/00_模拟数据_全策略对比.py`，看到输出
- 再回头读"01_架构"，对照代码看每个模块的位置
- 最后读策略详解时，**对照源码看，每行都问"为什么这么写"**
- 自己改一个参数（如双均线 5/20 → 10/30），看指标怎么变，培养手感
