## 学习建议

- 真小白: 从 [00 章](./00_写给零基础的你/00_读我.md) 开始, 跑通示例 + 看懂报告再读其他章
- 有基础: 直接 01 → 07 顺序读, 边读边跑 `examples/`
- 求职导向: 重点啃 06 (ML) + 07 (因子研究方法论)
- 自己改一个参数（如双均线 5/20 → 10/30），看指标怎么变，培养手感
