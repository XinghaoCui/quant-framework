# 价值因子详解

## 思想

> 买便宜的股票，卖贵的股票。

"便宜"和"贵"用**估值比率**衡量：PE、PB、PS、EV/EBITDA。

## 学术起源

**Fama & French (1992)** "The Cross-Section of Expected Stock Returns" 把价值因子（HML, High Minus Low BM）确立为解释股票收益的核心因子之一。

后续：
- 巴菲特的 alpha 大部分被 Fama-French 三因子 + 质量因子解释（Frazzini et al., 2018）
- 长期看（30 年），价值股年化超额收益约 3-5%
- 但 2010-2020 价值因子失效了 10 年（"价值之死"），2022 年才回归

## 价值因子的多种定义

| 因子 | 计算 | 含义 |
|---|---|---|
| BM (Book-to-Market) | 净资产 / 市值 | Fama-French 经典 |
| E/P | 利润 / 市值 | PE 的倒数 |
| S/P | 营收 / 市值 | 适合周期股 |
| CF/P | 现金流 / 市值 | 抗利润操纵 |
| EV/EBITDA 倒数 | EBITDA / EV | 考虑资本结构 |
| 反转 | -过去 1 年收益 | 价格代理 |

**最稳健的做法**：把多个估值因子合成一个"综合价值分"。

## 为什么会有价值溢价

### 1. 风险溢价说
- 价值股通常是"困境企业"或周期底部
- 风险高，所以要求额外回报

### 2. 行为金融说
- 投资者过度悲观，把基本面好的便宜股压得太便宜
- 高成长股被过度追捧

### 3. 限制套利说
- 价值股流动性差、做空成本高
- 套利者难以拉平估值差

## 价值因子的"漫长冬天"

2010-2020 年价值因子在美股几乎不赚钱：
- 利率长期下行，长久期资产（成长股）受益
- 互联网/科技股盈利集中，估值越高越正确
- 量化策略普及，套利者拉平了部分价值溢价

教训：
- 任何因子都可能失效十年
- 因子投资需要长期主义 + 风险控制
- 不能 all-in 单一因子

## 简化版实现：用反转作为价值代理

完整价值因子需要财报数据（PE、PB），本框架数据层暂时只有价格。
所以 `src/strategies/value_factor.py` 用一个**简化代理**：

$$
\text{Value-Proxy}_t = -\left(\frac{P_t}{P_{t-252}} - 1\right)
$$

即"过去 1 年跌得多 = 现在便宜"。

### 这个代理对吗？

部分对，部分错：

**对的地方**：
- 长期看，跌多了的股票确实更便宜（PB 下降）
- 长期反转效应客观存在（DeBondt-Thaler 1985）

**错的地方**：
- 跌可能是因为基本面恶化（盈利下降，PE 没变低）
- 价值股是"低估值"，不是"近期下跌"
- 真正的价值股可能 10 年没怎么动，但 PE 一直低

**结论**：本框架的 `ValueFactorStrategy` 主要用作**因子框架展示**。
真正做价值策略要接基本面数据。

## 本框架的实现

```python
class ValueFactorStrategy(Strategy):
    def __init__(self, lookback_days=252, top_pct=0.3, rebalance_days=21):
        ...

    def generate_weights(self, prices):
        past_return = prices / prices.shift(self.lookback_days) - 1.0
        value_score = -past_return  # 跌多 = 高分

        weights = pd.DataFrame(0.0, ...)
        for date in rebalance_dates:
            scores = value_score.loc[date].dropna()
            cheap = scores.nlargest(top_pct * n).index
            weights.loc[date, cheap] = 1.0 / n_top

        return weights.ffill()
```

## 改进方向（接入真实价值数据后）

### 1. 多因子综合价值分
```python
score = (z_score(1/PE) + z_score(1/PB) + z_score(1/PS)) / 3
```

### 2. 行业中性化
不同行业的"正常 PE"差异巨大（科技股 30 倍 vs 银行股 5 倍）。
在行业内排序更公平。

### 3. 价值陷阱过滤
低估值 + 高质量（ROE 高）才是真便宜。
单看 PE 容易买到"持续衰退"的伪价值。

### 4. 与动量结合
"便宜 + 趋势启动"是经典组合（Asness "Devil in HML's Details" 2013）。

## 跟动量因子的关系

价值和动量有**负相关**：
- 价值股 = 跌过的股票
- 动量股 = 涨过的股票

逻辑上是对立的，但实证上**长期来看两者都有 alpha**，
组合后波动率比单一因子低很多——这是经典的"价值+动量"60/40 配置思路。
