# 双均线策略详解

## 思想

```
短期均线（如 20 日）反映"近期均价"
长期均线（如 60 日）反映"较长期均价"

短上穿长（金叉）→ 近期价格在加速 → 上涨趋势确立 → 买入
短下穿长（死叉）→ 近期价格在减速 → 下跌趋势确立 → 卖出
```

## 数学

设价格序列 $P_t$，

$$
SMA_n(t) = \frac{1}{n} \sum_{i=0}^{n-1} P_{t-i}
$$

信号：

$$
\text{signal}_t = \begin{cases}
+1 & SMA_{20}(t) > SMA_{60}(t) \quad (\text{金叉持有期}) \\
-1 & SMA_{20}(t) < SMA_{60}(t) \quad (\text{死叉持有期})
\end{cases}
$$

## 为什么"理论上"有效

均线是**低通滤波器**：把价格的高频噪声滤掉，留下低频趋势。
当两条不同频率的滤波器交叉时，意味着"主导节奏"发生切换。

如果市场存在动量效应（趋势会延续），均线交叉就是个有效的"趋势启动"信号。

## 实证表现

历史上：
- 在**长牛市**（如 2010s 的美股、2015 前的 A 股小盘）双均线表现不错
- 在**震荡市**（如 2018-2019 美股、2014-2015 A 股大盘）表现差
- 总体上**未扣成本**有正期望，**扣完成本**接近零

经验上：
- 美股 SPY 上跑 50/200 日均线（"黄金交叉"系统），1990 年以来年化约 8%，最大回撤约 30%，比买入持有还差
- 真正能赚钱的双均线策略通常需要加**过滤器**（波动率、ADX、市场状态）

## 关键参数

### fast / slow 窗口选择

- 快慢比例通常 3:1 到 1:1
- 20/60 适合中频日线
- 5/20 适合短线（噪声多，信号差）
- 50/200 适合长线（信号慢，错过早期机会）

### 为什么不选 1/5 这种极短组合？

- 单日波动作为信号，几乎全是噪声
- 换手率爆炸，交易成本吃光超额收益

### 为什么不选 200/500 这种极长组合？

- 信号滞后到没意义（趋势已经走完才进场）
- 一旦反转，回撤极大

### 调参的陷阱

如果你"试 100 个参数组合，选最好的那个"，那是**过拟合**。
样本外大概率失效。

正确做法：
- 固定参数（如经典 20/60），在不同市场不同时段稳健测试
- 或者用 walk-forward：每年用过去 5 年数据选参数，下一年用这个参数

## 本框架的实现

`src/strategies/sma_cross.py`：

```python
class SmaCrossStrategy(Strategy):
    def __init__(self, fast=20, slow=60, allow_short=False):
        ...

    def generate_weights(self, prices):
        fast_ma = prices.rolling(self.fast).mean()
        slow_ma = prices.rolling(self.slow).mean()

        signal = pd.DataFrame(0.0, ...)
        signal[fast_ma > slow_ma] = 1.0
        if self.allow_short:
            signal[fast_ma < slow_ma] = -1.0

        # 等权分配
        n_active = signal.abs().sum(axis=1)
        weights = signal.div(n_active.replace(0, 1), axis=0)
        return weights
```

注意：
- 输出是 DataFrame（每天对每只票一个权重），不是单标的的 +1/-1 信号
- 权重做了归一化，保证总权重 ≤ 1（不加杠杆）
- 引擎层自动 shift(1) 防未来函数

## 改进方向

### 1. 加趋势强度过滤
ADX > 25 时才执行金叉信号，避免震荡市。

### 2. 加波动率过滤
波动率高时降低仓位（"危险时少做"）。

### 3. 三均线
20/60/200。短中长全部对齐才进场。

### 4. 用 EMA 替换 SMA
EMA 对近期价格更敏感，信号略快。

### 5. 加止损
跌破入场价 2×ATR 平仓。

每个改进都对应代码上"在 generate_weights 里加一行过滤"。
本框架的设计让这些扩展很容易加。

## 跟动量因子的关系

双均线本质是**时序动量**：
> "我自己的价格在上涨" → 买自己

横截面动量是：
> "我比别人涨得多" → 买相对赢家

两者背后是同一个市场效应（动量），但实现方式不同。
横截面动量在统计上更稳定（多空对冲了系统性风险）。
