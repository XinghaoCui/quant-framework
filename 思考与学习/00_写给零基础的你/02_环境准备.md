# 环境准备

> 10 分钟, 手把手装好运行环境。
> 假设你**完全没装过 Python**, 跟着一步步做就行。

## 准备时间和心理预期

- **大约耗时**: 10~30 分钟（取决于下载速度）
- **会装什么**: Python、几个数据科学库、Git
- **不会装什么**: 不需要数据库、不需要 Docker、不需要付费软件

如果你已经装过 Python 和 pandas, 直接看末尾的 [快速通道](#已有-python-环境的快速通道)。

## 一、安装 Python

### Windows
1. 去 https://www.python.org/downloads/ 下载 **Python 3.11 或更新版**（本仓库用 3.14, 但 3.11+ 都行）
2. **重要**: 安装时勾选 ☑ "**Add Python to PATH**"（不勾后面会很麻烦）
3. 一路 Next 装完

### Mac
推荐用 Homebrew:
```bash
# 装 Homebrew（如果没装过）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 装 Python
brew install python@3.11
```

### 验证
打开命令行（Windows 是 PowerShell 或 cmd, Mac 是 Terminal）, 输入:
```bash
python --version
```
看到 `Python 3.x.x` 就行。

如果提示 "command not found", 说明 PATH 没设好, 回去重装并勾上那个选项。

## 二、安装 Git（拿代码用的）

### Windows
去 https://git-scm.com/download/win 下载安装, 一路 Next。

### Mac
```bash
brew install git
```

### 验证
```bash
git --version
```
看到 `git version 2.x.x` 就行。

## 三、克隆本仓库

打开命令行, 进入你想放代码的目录（比如桌面）:
```bash
# Windows
cd Desktop

# Mac
cd ~/Desktop
```

然后克隆:
```bash
git clone https://github.com/XinghaoCui/quant-framework.git
```

> ⚠️ 这是个**私有仓库**, 直接 clone 会让你输 GitHub 账号密码。
> 如果你只是想阅读, 推荐 fork 一份到自己的账号下, 或者让作者邀请你做协作者。
> 也可以在 GitHub 网页上直接 Download ZIP。

如果你卡在网速或代理问题, 看 [06 常见报错](./06_常见报错与FAQ.md#github-克隆很慢或失败)。

进入项目目录:
```bash
cd quant-framework
```

## 四、（强烈推荐）创建虚拟环境

虚拟环境 = 给这个项目单独建一个"小盒子", 装的库不会污染你电脑上其他项目。
**这是 Python 圈的标准做法, 不做以后会乱套**。

```bash
# 在项目根目录下
python -m venv .venv

# 激活虚拟环境
# Windows (PowerShell):
.venv\Scripts\Activate.ps1

# Windows (cmd):
.venv\Scripts\activate.bat

# Mac / Linux:
source .venv/bin/activate
```

激活成功后, 命令行最前面会多一个 `(.venv)` 的标识。

> 如果 PowerShell 提示"无法加载脚本", 是 Windows 的执行策略问题。
> 用管理员身份开 PowerShell, 跑一次 `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`, 然后选 Y。

## 五、装依赖库

```bash
pip install -r requirements.txt
```

这一步会装:
- `pandas` — 操作表格数据（最核心）
- `numpy` — 数值计算
- `matplotlib` — 画图
- `scipy` — 科学计算
- `scikit-learn` — 机器学习
- `yfinance` — 抓美股数据
- `akshare` — 抓 A 股数据

大概要下几十 MB, 一两分钟。

> 国内网速慢的话, 用清华镜像加速:
> ```bash
> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
> ```

## 六、验证装好了

跑一行 Python:
```bash
python -c "import pandas, numpy, sklearn; print('OK')"
```
打出 `OK` 就成了。

如果报 `ModuleNotFoundError`, 说明虚拟环境没激活, 或者 pip 装到了别处。
重新激活虚拟环境再装一遍。

## 七、（可选）装个像样的编辑器

命令行里写代码很难受, 推荐装一个**VS Code**:
- 下载: https://code.visualstudio.com/
- 装完打开仓库目录: `File → Open Folder → 选 quant-framework`
- 装个扩展: `Python` (微软出的那个), 一切就丝滑了

## 已有 Python 环境的快速通道

跳过上面所有步骤, 三行命令搞定:
```bash
git clone https://github.com/XinghaoCui/quant-framework.git
cd quant-framework
pip install -r requirements.txt
```
