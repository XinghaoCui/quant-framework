# 本仓库阅读地图

> 5 分钟看清: 这个仓库一共有什么, 你应该按什么顺序读, 哪些可以跳。
> 这一篇是整个仓库的**导航总图**, 建议读完后**收藏**, 以后随时回来查。

## 一、仓库长什么样

```
量化/
├─ src/                   ← 代码（你以后会改的地方）
├─ examples/              ← 跑得起来的示例（你今天已经跑过 examples/00 了）
├─ tests/                 ← 单元测试
├─ 思考与学习/             ← 中文学习文档（你正在读的这部分）
│   ├─ 00_写给零基础的你/    ← 你在这里
│   ├─ 01_架构与设计/       ← 仓库怎么组织
│   ├─ 02_核心术语/         ← 术语表
│   ├─ 03_因子与策略/       ← 策略和因子讲解
│   ├─ 04_回测指标解读/      ← 报告里每个指标
│   ├─ 05_扩展指南/         ← 如何自己加策略/因子
│   ├─ 06_机器学习入门/      ← ML 在量化里的应用
│   ├─ 07_因子研究方法论/    ← 怎么科学研究一个因子
│   ├─ 08_组合优化/          ← 选完股票后怎么分仓位
│   └─ 09_基本面数据/        ← 价格之外的第二条信息流: 公司财务
├─ reports/               ← 跑出来的图和 csv
└─ README.md
```

**学习路径就是按编号读 `思考与学习/`**, 配合 `examples/` 跑跑代码。

## 二、按你的目标选路线

### 路线 A: 我只想"看看量化是啥, 满足好奇" (3 小时)

```
00 写给零基础的你 (你正在这里, 全读)
   ↓
03 因子与策略 / 什么是因子.md (只读这一篇)
   ↓
跑 examples/00 (改改参数玩玩)
   ↓
结束。你已经超过 95% 没动手的人。
```

### 路线 B: 我想"系统学量化, 为求职/深造做准备" (1~2 周)

```
00 写给零基础的你 (全读, 边读边跑示例)
   ↓
01 架构与设计 (全读, 了解仓库设计哲学)
   ↓
02 核心术语 (查阅式, 不需要背)
   ↓
03 因子与策略 (全读, 重点理解"什么是因子")
   ↓
跑 examples/01~03 (真实数据)
   ↓
04 回测指标解读 (全读, 反复回头查)
   ↓
05 扩展指南 (动手新增一个自己想的策略)
   ↓
07 因子研究方法论 (全读, 跑 examples/20~23)  ★ 求职核心
   ↓
06 机器学习入门 (全读, 跑 examples/10~13)    ★ ML 量化求职核心
   ↓
08 组合优化 (全读, 跑 examples/30~32)       ★ 完成"研究-实现-评估" 闭环
   ↓
09 基本面数据 (全读, 跑 examples/40~43)     ★ 接入第二条信息流, 全栈完成
   ↓
你现在具备初级量化研究员的认知结构了。
```

### 路线 C: 我有编程基础, 想直接上代码 (3~5 天)

```
00 写给零基础的你 / 05 阅读地图 (你正在这里, 只读 03/04 节奏)
   ↓
01 架构与设计 / 为什么这样分层.md (10 分钟)
   ↓
直接读 src/ 里所有代码 + 跑 examples/ 全部
   ↓
对着 06/07 章查不懂的细节
```

### 路线 D: 我已经会量化, 想看看这个仓库有没有用 (1 小时)

```
README.md (5 分钟)
   ↓
src/factors/evaluation.py + src/strategies/multi_factor.py (15 分钟)
   ↓
src/ml/walkforward.py (15 分钟)
   ↓
07 因子研究方法论 / 06 完整因子研究流程.md (15 分钟)
   ↓
判断有用没用。
```

## 三、每一章一句话总结

| 章 | 主题 | 一句话 | 难度 | 时长 |
|---|---|---|---|---|
| 00 | 写给零基础的你 | 把你从 0 拉到 "跑过回测、看懂报告" | 🟢 入门 | 1 小时 |
| 01 | 架构与设计 | 仓库为什么这样组织、向量化回测原理 | 🟢 入门 | 30 分钟 |
| 02 | 核心术语 | 30+ 量化术语速查 + 收益率定义 | 🟢 查阅 | 散读 |
| 03 | 因子与策略 | 因子是什么、本仓库三个经典策略详解 | 🟡 中等 | 1 小时 |
| 04 | 回测指标解读 | 夏普 / 回撤 / 卡玛等指标的数学和经验 | 🟡 中等 | 30 分钟 |
| 05 | 扩展指南 | 怎么新增策略 / 因子 / 排查数据问题 | 🟡 中等 | 30 分钟 |
| 06 | 机器学习入门 | ML 在量化怎么用、防过拟合、滚动重训 | 🔴 进阶 | 3-5 小时 |
| 07 | 因子研究方法论 | IC/IR/分组/衰减/多因子合成完整方法论 | 🔴 进阶 | 3-5 小时 |
| 08 | 组合优化 | 最小方差/风险平价/均值方差/有效前沿、约束与稳健性 | 🔴 进阶 | 2-4 小时 |
| 09 | 基本面数据 | 财报三大表、防未来函数、价值/质量/成长/安全 4 类因子、技术+基本面整合 | 🔴 进阶 | 2-3 小时 |

> 🟢 = 没编程/金融基础也能读
> 🟡 = 需要会基本 Python 和懂基本"股票收益"概念
> 🔴 = 需要会 pandas/numpy, 看得懂简单公式

## 四、示例与文档的对应关系

文档不是孤立的, **每一篇都有配套示例**:

| 文档章节 | 配套示例 | 怎么用 |
|---|---|---|
| 00 章 | `examples/00_模拟数据_全策略对比.py` | 第一次跑这个就行 |
| 03 章 | `examples/01_双均线_美股.py` `02_动量_美股.py` `03_价值因子_A股.py` | 读完策略详解配着跑 |
| 06 章 | `examples/10~13_ML*.py` | 读完一篇 ML 文档就跑一个 |
| 07 章 | `examples/20~23_*.py` | 读完因子研究文档跑这些 |
| 08 章 | `examples/30~32_*.py` | 读完组合优化文档跑这些 |
| 09 章 | `examples/40~43_*.py` | 读完基本面文档跑这些 |

**强烈推荐边读边跑**, 比纯读快 3 倍。

## 五、代码模块对照

`src/` 下每个目录都对应一类工作:

| 目录 | 干嘛的 | 对应文档 |
|---|---|---|
| `src/data/` | 抓数据（美股/A股/模拟 + 基本面） | 05 / 数据源问题排查、09 |
| `src/engine/` | 跑回测、算指标 | 01 / 向量化回测原理 |
| `src/strategies/` | 策略类 (含 OptimizedStrategy) | 03 / 各策略详解、08 |
| `src/factors/` | 因子计算 + 因子评估 + 基本面因子 | 03 / 07 / 09 |
| `src/ml/` | 机器学习全套 | 06 |
| `src/portfolio/` | 组合优化器 (min_var / risk_parity / mean_var / 有效前沿) | 08 |
| `src/utils/` | 画图 + 报告输出 | (无专章) |

## 八、学完一遍之后

如果你按路线 B 走完了, 接下来:
1. 自己想一个因子（可以参考论文也可以自己拍脑袋）, 用 07 章的方法论评估它
2. 把它加入 multi-factor 组合, 看是否能跑赢现有
3. 把整个研究过程写成一篇文章, 这就是**初级量化研究员的标准产出**

这个仓库不是终点, **是你自己的量化研究的起点**。
