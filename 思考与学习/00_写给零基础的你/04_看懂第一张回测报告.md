# 看懂第一张回测报告

> 20 分钟, 把上一篇看到的那一堆数字逐条讲清。
> 读完, 那张报告对你不再是天书。

## 我们要解释的就是这张表

```
========================================
  累计收益        :    202.92%
  年化收益        :     16.52%
  年化波动率       :      9.28%
  夏普比率        :      1.694
  Sortino比率   :      1.735
  最大回撤        :    -16.37%
  Calmar比率    :      1.009
  胜率          :     53.09%
  盈亏比         :      1.081
  交易天数        :       1827
  平均换手率(单边)   :      0.032
  总交易成本       :      0.046
========================================
```

11 个数字。一个一个讲清。
读完你就拥有了看懂任何量化报告的基础能力。

## 一、累计收益 (Cumulative Return) = 202.92%

> 你的 1 块钱, 7 年后变成了 3.03 块。

最直观的指标:
```
累计收益 = (最终净值 - 初始净值) / 初始净值
       = (3.03 - 1) / 1
       = 2.03
       = 202%
```

但累计收益是最容易骗人的:
- 同样涨 200%, 一年涨完 vs 二十年涨完, 完全两回事
- 看累计收益没法跨策略比

所以下一项才是真正有用的:

## 二、年化收益 (Annual Return) = 16.52%

> 平均每年涨 16.52%。

把累计收益换算成"每年涨多少", 才能跨周期比:
```
年化收益 = (1 + 累计收益)^(1/年数) - 1
       = (1 + 2.029)^(1/7.25) - 1
       = 0.165
       = 16.52%
```

经验判读:
| 年化收益 | 评级 | 参考 |
|---|---|---|
| < 5% | 不如存款 | 银行理财都 3-4% |
| 5-10% | 一般 | 大致跟得上指数 |
| 10-20% | 不错 | 已经跑赢大多数主动基金 |
| 20-30% | 优秀 | 接近顶级公募 |
| > 50% | 警惕 | 大概率有 bug 或过拟合, 先查代码 |

> 看到 100%+ 年化的第一反应不应该是兴奋, 是怀疑。
> 真实世界里能长期年化 30% 的人都是大师。

## 三、年化波动率 (Annual Volatility) = 9.28%

> 你的账户每年的"心跳幅度"是 9.28%。

简单说就是净值波动有多大:
- 波动率小 = 净值像一条直线, 心脏好
- 波动率大 = 净值像过山车, 心脏不好

波动率本身不分好坏, 关键是"波动是否换来了相应的收益"。
这就引出了下一个指标:

## 四、夏普比率 (Sharpe Ratio) = 1.694

> 量化里最重要的一个指标。每承担 1 单位风险, 你拿到多少超额收益。

公式:
```
夏普 = (年化收益 - 无风险收益率) / 年化波动率
    = (16.52% - 0%) / 9.28%
    ≈ 1.78  (本框架默认无风险率 = 0, 真实算出 1.694 有日频近似差异)
```

经验判读 (这张表你以后会用一辈子):
| 夏普 | 评级 |
|---|---|
| < 0 | 亏钱 |
| 0 ~ 0.5 | 不如躺平 |
| 0.5 ~ 1.0 | 一般 |
| 1.0 ~ 1.5 | 不错 |
| 1.5 ~ 2.0 | 优秀 (主流公募水平) |
| 2.0 ~ 3.0 | 顶级 |
| > 3.0 | 警惕, 99% 是过拟合或 bug |

> 一个夏普 2.0 的策略 = 平均每承受 1% 波动, 能换 2% 收益。
> 一个夏普 0.5 的策略 = 同样承受 1% 波动, 只换 0.5% 收益。
> 后者还不如把钱放余额宝。

## 五、Sortino 比率 = 1.735

> 夏普的"只罚下跌"版本。

夏普的缺点: 把"涨太多" 也算成波动。 但你不在乎涨太多, 你在乎跌太多。
Sortino 只算下跌方向的波动:

```
Sortino = (年化收益 - 无风险率) / 下跌波动率
```

判读跟夏普一样, 通常 Sortino 比夏普高一点。
夏普和 Sortino 差距越大, 说明策略"涨得猛跌得轻", 是好事。

## 六、最大回撤 (Max Drawdown) = -16.37%

> 从历史最高点跌到最低点, 你最多亏过 16.37%。

这是另一个极其重要的指标, 衡量你最痛苦的时刻有多痛苦。

为什么重要? 因为人的心理承受能力是有限的:
- 账户跌 10%, 大多数人能扛
- 跌 30%, 一半人会割肉
- 跌 50%, 几乎所有人会割肉退出

> 一个夏普 2.0 但回撤 -60% 的策略, 还不如夏普 1.0 但回撤 -10% 的策略, 因为前者你根本扛不到回本。

经验判读:
| 最大回撤 | 评级 |
|---|---|
| > -10% | 非常稳, 适合大资金 |
| -10% ~ -20% | 稳, 主流公募水平 |
| -20% ~ -30% | 一般, 心脏要好 |
| -30% ~ -50% | 风险大 |
| < -50% | 你扛不住的 |

## 七、Calmar 比率 = 1.009

> 年化收益 / |最大回撤|, 衡量"收益相对于最大痛点的比值"。

```
Calmar = 16.52% / 16.37% ≈ 1.01
```

直觉是: 你忍一年最深的痛, 能换回相当于这个痛的多少倍收益?
- Calmar > 1: 痛得值
- Calmar > 2: 很值
- Calmar > 3: 极值

Calmar 是夏普的好补充, 因为夏普用的是"平均波动", Calmar 用的是"最痛的那次"。
对个人投资者, Calmar 比夏普更接近"我能不能扛住"的直觉。

## 八、胜率 (Win Rate) = 53.09%

> 赚钱的天数 / 总交易天数 = 53.09%

经常听到"我这策略胜率 80%"听着很厉害, 但胜率高不等于赚钱:

```
情景 A: 胜率 80%, 但每次输 -10%, 赢 +1%
       100 次交易: 80 × 1% + 20 × (-10%) = 80% - 200% = -120%, 巨亏

情景 B: 胜率 30%, 但每次输 -1%, 赢 +10%
       100 次交易: 30 × 10% + 70 × (-1%) = 300% - 70% = 230%, 大赚
```

胜率必须**配合下一个指标看**:

## 九、盈亏比 (Profit/Loss Ratio) = 1.081

> 平均一次赚的钱 / 平均一次亏的钱

```
盈亏比 = (平均盈利天的收益) / |平均亏损天的损失|
       = 1.08
```

用 "胜率 × (1 + 盈亏比) - 1" 大致能判断期望收益的方向:
- 胜率 50% + 盈亏比 1.0 = 不赚不亏
- 胜率 50% + 盈亏比 1.5 = 长期赚
- 胜率 70% + 盈亏比 0.8 = 长期不赚

我们这个 SmaCross: 胜率 53% + 盈亏比 1.08, 略微正期望, 整体小赚。

## 十、交易天数 (Trading Days) = 1827

> 回测覆盖了 1827 个交易日 ≈ 7.25 年。

样本越长越可信。少于 1 年的回测基本没有统计意义, 任何指标都可能是运气。

## 十一、平均换手率 (单边) = 0.032 (3.2%)

> 平均每天换 3.2% 的仓位（单边）。

换手率 = 调仓激烈程度:
- < 1% (日): 极低频, 适合大资金
- 1-5%: 中低频, 月度级别调仓
- 5-20%: 中高频, 周度调仓
- > 20%: 高频, 几乎天天动

换手率高不一定错, 但意味着交易成本会吃掉大量收益。
这就引出最后一个指标:

## 十二、总交易成本 = 0.046 (4.6%)

> 7 年里, 你为交易付出的总成本 = 累计 4.6%。

来源:
- 佣金: 买卖股票要给券商的钱（A 股双边万 3）
- 滑点: 想买的价跟实际成交价的差
- 印花税（A 股卖出 1‰）

注意这个 4.6% 已经从"累计收益 202.92%" 中扣掉了, 不是再扣一次。

直觉: 一个看着夏普 3.0 的高频策略, 把成本算进去可能只剩夏普 0.5。
回测一定要含成本, 否则就是骗自己。本框架默认含, 这是基本姿势。

## 总结表（建议背下来）

重要性那一列用星数表示，越多越重要。

| 指标 | 经验判读 | 重要性 |
|---|---|---|
| 年化收益 | 10-20% 不错, >50% 警惕 | 中 |
| 夏普 | 1-2 优秀, >3 警惕 | 高 |
| Sortino | 通常略高于夏普 | 低 |
| 最大回撤 | -10% 内稳, -30%+ 心脏要好 | 高 |
| Calmar | >1 痛得值, >2 很值 | 中 |
| 胜率 | 50% 上下都正常 | 低 (必须配盈亏比) |
| 盈亏比 | 配胜率一起看才有意义 | 低 |
| 换手率 | 看成本影响 | 低 |
| 总交易成本 | 越低越好, 但策略类型决定下限 | 低 |

## 看回 03 那张对照表

```
                 年化收益      夏普比率      最大回撤   Calmar比率
SmaCross     16.52%       1.69         -16.37%   1.01
Momentum     39.85%       1.73         -19.97%   1.99
ValueFactor  13.26%       0.79         -32.58%   0.41
```

现在你能解读了:
- Momentum: 年化 40%, 夏普 1.73, 回撤 20%, Calmar 接近 2, 全面胜出
- SmaCross: 年化 16%, 夏普 1.69, 回撤 16%, 慢但稳, 心脏舒服
- ValueFactor: 年化 13% 夏普 0.79 回撤 32%, Calmar 0.41, 这次表现最差

但这只是模拟数据上的结果。真实数据上结果可能完全不同, 不能下结论"动量永远最好"。

## 最重要的一句话

> 没有"完美策略", 只有"适合自己风险偏好的策略"。

一个年化 100% 但回撤 -70% 的策略, 大多数人扛不住, 不适合普通人。
一个年化 8% 但回撤 -5% 的策略, 对追求"稳"的资金完美。

学会看报告, 是为了让你找到适合自己的策略, 不是找最"花哨"的那个。

## 一句话总结

> 夏普看收益效率, 回撤看心理承受, Calmar 看划不划算, 三者足以打分一个策略 80%。
