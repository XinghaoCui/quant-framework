# 分组回测与单调性

IC 看的是线性相关, 但 IC=0.05 也可能是"5% 的票贡献了全部信号, 其他 95% 是噪声"。
**分组回测**直接看 "按因子分组后, 高组真的跑赢低组吗", 提供另一个角度的硬证据。

## 一、分组回测怎么做

四步:

```
1. 每天 t, 把所有股票按因子值 f(t) 排序
2. 分成 N 组 (常用 N=5 或 N=10), 标记 Q1 (最低) ... Qn (最高)
3. 计算每组在未来 horizon 天内的等权平均收益
4. 跨日把每组的收益累加 → 每组一条净值曲线
```

如果因子有效, 你会看到:

```
净值
 ↑                                  ━━━━ Q5 (最高分组)
 |                            ━━━━
 |                      ━━━━        ━━━━ Q4
 |                ━━━━              ━━━━ Q3
 |          ━━━━                    ━━━━ Q2
 |   ━━━━                          ━━━━ Q1 (最低分组)
 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━→  时间
```

**单调性 (Q5 > Q4 > Q3 > Q2 > Q1) 是因子有效的硬证据。**

## 二、为什么分组比 IC 更直观

IC 是个数字, 但分组回测能看到很多信息:

### 1. 单调 vs 双峰
有些"因子"看着 IC 不错, 但分组其实是"两头好中间差":
```
Q1: +5%   Q2: -2%   Q3: -3%   Q4: -1%   Q5: +6%
```
这说明因子捕捉的不是"线性预测力", 而是"极端值的特殊行为"。
IC 算出来不一定低, 但实际选股逻辑很危险。

### 2. 多头驱动 vs 空头驱动
```
情景 A: Q1 = -10%, Q5 = +2% (空头驱动, 低分股票特别差)
情景 B: Q1 = -2%,  Q5 = +10% (多头驱动, 高分股票特别好)
```
A 在 A 股做不了 (难做空), 即使 IC 看着不错也用不了;
B 适合纯多头策略。

> A 股做不了空, 因子能否纯多头实战, 看 Q5 是不是显著跑赢中位组。

### 3. 多空价差 (Long-Short Spread)
Qn - Q1 的差异, 是"纯因子 alpha", 剔除了市场 beta:

```python
长空收益 = Q5 收益 - Q1 收益
```

这是因子研究的核心指标之一。年化夏普 > 1 的多空价差就是不错的因子。

## 三、N 怎么选

- **N=5 (五分位)**: 最常用, 每组样本足够, 信号清晰
- **N=10 (十分位)**: 标的池大时用 (>200 只), 看极端组的表现
- **N=20+**: 太细, 每组样本太少, 噪声大

> 中国 A 股可投资标的池约 3000~4000 只, 用 N=10 没问题。
> 但如果你只有 30 只股票, N=5 已经偏紧, 不要再细分。

## 四、horizon 怎么选

horizon (持有天数) 影响很大:

- **horizon=1**: 信号最锐利, 但换手频繁, 实盘成本高
- **horizon=5~20**: 大多数研究用的范围, 平衡信号和成本
- **horizon=60+**: 低频策略, 适合大资金, 但反应慢

不同 horizon 下的分组结果对比, 实际上就是 IC 衰减分析 (下一篇)。

## 六、注意: 分组回测不等于策略回测

`quantile_backtest` 输出的"年化收益 / 夏普"是**理论估计**, 不是可执行策略的真实表现:

- 它假设"每天等权持有这一组, 持有 horizon 天, 然后换"
- 没有考虑实际调仓周期、换手率、交易成本
- 不同 horizon 下的"日均收益"也是近似折算 (除以 horizon)

要算真实可执行策略的表现, 应该:
1. 用 `MultiFactorStrategy` 或自己写策略生成 weights
2. 丢给 `Backtester` 跑真实回测 (含成本)

分组回测是**研究工具**, 不是**生产工具**。它的价值是快速判断因子有没有单调性、值不值得做策略。

## 七、单调性测试 (进阶)

严格的统计检验有几种:

### 1. Spearman 排名一致性
对 [Q1 年化收益, ..., Qn 年化收益] 跟 [1, 2, ..., n] 做 Spearman:
完美单调 → 1.0; 完全乱 → 0; 反向单调 → -1.0。

### 2. 多空价差的 t 统计
$$
t_{LS} = \frac{\overline{Q_n - Q_1}}{\sigma(Q_n - Q_1)} \times \sqrt{N}
$$
和上一篇的 t 统计一样, |t| > 2 算显著。

### 3. Newey-West 修正
因为 horizon > 1 时收益序列有自相关, 标准误要做 Newey-West HAC 修正才严谨。
正经的因子研究报告通常要给这个修正后的 t 统计。
本框架暂时不做, 后面会扩展。

## 八、小结

- 分组回测 = 把股票按因子分 N 组, 看每组的未来收益
- 看三件事: **单调性**、**多空价差**、**多头/空头是哪边驱动**
- N=5 是默认, 标的少时不要细分
- 这是研究工具, 不是策略回测; 真实策略要走 `Backtester`
- 与 IC 互补: IC 看线性相关, 分组看单调性, 都过关才放心
