# 因子研究在做什么

> 📚 前置: 已经读过 03 章因子、04 章指标。 完全零基础先看 [00 章](../00_写给零基础的你/00_读我.md)。

第 3 章讲了"因子是什么、有哪些"。这一章要回答更核心的问题:
**拿到一个因子, 量化研究员到底每天在做什么?**

## 一、错误的姿势

90% 的入门者拿到因子做的事是:

```
计算因子 → 选 top 30% → 跑回测 → 看夏普
```

跑出来夏普 2.3 就发朋友圈, 跑出来夏普 0.5 就换下一个因子。这是**赌博**, 不是研究。

为什么这是错的:
- 一次回测的夏普完全可能是运气, 没有统计意义
- 不知道因子**为什么**有效 / 失效, 没法迭代
- 没法把多个弱因子合成强信号 (业界标准做法)
- 真实实盘里因子会衰减, 你完全没准备

## 二、正确的姿势: 因子研究四问

量化研究员每天在做的, 是用一套**标准化流程**回答四个问题:

### 1. 有用吗?
今天的因子值能不能预测未来收益? → **IC / Rank IC**

> "信息系数 (Information Coefficient)" 是量化界的通用语言。
> 横截面 IC = 在某一天, 所有股票的因子值排序与未来收益排序的相关系数。
> 把所有日子的 IC 取均值, 就是这个因子的整体预测力。

### 2. 稳不稳?
IC 是稳定为正, 还是时正时负忽大忽小? → **IR (信息比率) / t 统计**

> 一个 IC 均值 0.05 但每天乱跳的因子, 不如一个 IC 均值 0.03 但每天稳定为正的因子。
> 稳定性比绝对值更值钱, 因为稳定的信号能在实盘活下来。

### 3. 单调吗?
按因子值分成 5 组, 高组真的跑赢低组吗? → **分组回测 (Quantile Backtest)**

> 单调性 (Q5 > Q4 > Q3 > Q2 > Q1) 是因子有效的"硬证据"。
> IC 是线性相关, 可能被几只极端股票拉高;
> 单调性看的是分组排序, 更可靠。

### 4. 信号能维持多久?
1 天后 IC 高, 60 天后 IC 还在吗? → **IC 衰减曲线 (IC Decay)**

> 衰减快 → 高频换仓 → 交易成本吃掉收益, 实盘扛不住。
> 衰减慢 → 持仓几周到几个月, 适合大资金, 成本低。
> 衰减节奏决定了实盘的**调仓频率**, 这是工程上的关键决策。

**这四问构成了"单因子研究报告"的核心。** 任何一个因子在被采纳前都要过这一关。

## 三、研究完单因子之后

四问通过的因子还不是终点, 后面还要:

### 5. 多因子合成 (本章 05)
单因子 IC 通常 0.03~0.08, 信号弱。把几个**相关性低**的因子合成综合分:

```
综合分 = w1 × 标准化(动量) + w2 × 标准化(价值) + w3 × 标准化(质量)
```

加权方式有等权 / IC 加权 / ML 学习权重 (第 6 章 ML 模块讲的就是这个)。

### 6. 风险归因和组合优化 (下一章 08)
"选 top 30% 等权"只是个起点。真实组合还要考虑:
- 行业暴露 (别只买银行)
- 风格暴露 (别全是高动量)
- 风险预算 (低相关股票权重高一点)

## 四、为什么这一章值得反复读

因子研究是横截面量化的**方法论核心**。

无论你将来做:
- 传统多因子选股 → IC / 分组 / 合成
- 机器学习量化 → IC 是评估指标, 分组是验证手段
- 因子 ETF / Smart Beta → 报告里全是 IC 和分位数收益
- 私募研究员面试 → 必考 IC / IR / 分组

这套语言都是必备的。把这章吃透, 你跟一线量化研究员就能"说同一种话"了。
