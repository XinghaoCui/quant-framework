# 量化金融术语速查表

> 这一章是查阅式的, 不需要从头读到尾。 遇到不懂的术语来这里搜就行。
> 完全零基础先看 [00 章 / 写给零基础的你](../00_写给零基础的你/00_读我.md)。

按主题分组，每个术语一句话定义 + 必要的展开。

## 一、收益与风险

| 术语 | 英文 | 简明定义 |
|---|---|---|
| 收益率 | Return | 价格变化的百分比，`r = P_t/P_{t-1} - 1` |
| 对数收益率 | Log return | `ln(P_t/P_{t-1})`，可加可减，统计上更友好 |
| 累计收益 | Cumulative return | 一段时间总共赚了百分之几 |
| 年化收益 | Annualized return | 把不同长度时段折算成"等价的稳定年增长率" |
| 波动率 | Volatility | 收益率的标准差，衡量"剧烈程度" |
| 年化波动率 | Annualized vol | 日波动 × √252，因为方差按时间线性叠加 |
| 下行波动率 | Downside deviation | 只算亏损的标准差，更贴近"风险"直觉 |
| 最大回撤 | Max drawdown | 从历史高点到后续低点的最大跌幅 |
| VaR | Value at Risk | 在 95% 置信度下，最坏 1 天会亏多少 |
| CVaR / ES | Expected Shortfall | 最坏 5% 那部分日子的平均亏损 |

## 二、绩效比率

| 术语 | 公式 | 用途 |
|---|---|---|
| 夏普比率 | `(年化收益 - rf) / 年化波动` | 单位风险下的超额收益 |
| Sortino 比率 | `(年化收益 - rf) / 下行波动` | 不惩罚"上行波动" |
| Calmar 比率 | `年化收益 / |最大回撤|` | 收益相对最大痛点 |
| 信息比率 IR | `(策略 - 基准) 年化 / 跟踪误差` | 主动管理是否有 alpha |
| Treynor 比率 | `(收益 - rf) / β` | 衡量系统性风险溢价 |

## 三、因子与风险模型

| 术语 | 简明定义 |
|---|---|
| 因子 (Factor) | 对每只股票打分的某种数值，按这个分排序后高分组与低分组有系统性收益差异 |
| Alpha | 不能被市场（或风险因子）解释的超额收益 |
| Beta | 个股相对市场的敏感度，`β=1` 是与市场同步 |
| Fama-French 三因子 | 市场、规模 (SMB)、价值 (HML) 三个解释股票收益的因子 |
| Carhart 四因子 | 在三因子基础上加动量因子 (UMD) |
| 因子暴露 | 组合在某个因子上的敞口大小 |
| 因子中性 | 组合在某因子上的暴露 ≈ 0，比如"市值中性" |
| 多因子模型 | 把多个因子加权合成一个综合得分 |

## 四、交易与执行

| 术语 | 简明定义 |
|---|---|
| 仓位 (Position) | 当前持有的股票数量或比例 |
| 权重 (Weight) | 该资产占组合的比例，∑w ≤ 1 |
| 换手率 | 一段时间内交易额 / 组合规模，衡量交易频繁度 |
| 滑点 (Slippage) | 想要的成交价与实际成交价的差 |
| 冲击成本 | 大单推动价格本身造成的额外损失 |
| 调仓 (Rebalance) | 把组合调整到新的目标权重 |
| 持有期 (Holding period) | 一笔头寸从开到关的时间 |
| T+1 / T+0 | A 股是 T+1（今天买明天才能卖），美股是 T+0 |
| 做空 (Short) | 借股票卖出，赌价格下跌，跌了再买回还掉 |
| 杠杆 (Leverage) | 总头寸 / 自有资金，>1 即放大风险与收益 |

## 五、策略类型

| 类型 | 例子 |
|---|---|
| 趋势策略 (Trend-following) | 双均线、海龟、Donchian 通道 |
| 横截面策略 (Cross-sectional) | 动量、价值、质量等因子选股 |
| 均值回归 (Mean reversion) | 配对交易、布林带、短期反转 |
| 套利 (Arbitrage) | 跨市场、跨品种、ETF 折溢价 |
| 事件驱动 | 财报后漂移、并购套利 |
| 做市 (Market making) | 同时挂买卖盘赚价差 |
| 高频 (HFT) | 持仓时间秒级以下 |

## 六、统计与机器学习

| 术语 | 简明定义 |
|---|---|
| 平稳序列 | 均值方差不随时间变的时间序列 |
| 协整 | 两个非平稳序列的线性组合是平稳的（配对交易基础） |
| 自相关 | 序列与自己滞后值的相关性 |
| IC (信息系数) | 因子值与未来收益的相关系数（横截面） |
| Rank IC | 用秩相关代替皮尔逊相关，更稳健 |
| Turnover | 策略每期换仓的比例 |
| Look-ahead bias | 未来函数：用了未来才知道的信息 |
| Survivorship bias | 幸存者偏差：只看现在还在的票，忽略退市的票 |
| 过拟合 | 在样本内表现好，样本外失效 |
| Walk-forward | 滚动样本外验证 |

## 八、几个高频被搞混的概念

- **复权 vs 不复权**：用收盘价做回测必须用复权价，否则除权日会出现"假跌停"。
- **简单收益 vs 对数收益**：累乘用简单收益，加和/统计用对数收益。
- **波动率 vs 标准差**：通常指年化波动率，不是日标准差。
- **夏普 vs 信息比率**：夏普分母用绝对波动率，IR 分母用跟踪误差。
- **β vs 相关性**：β = cor(R_i, R_m) × σ_i / σ_m，相关性只是其中一部分。
